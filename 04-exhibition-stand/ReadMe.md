# Exhibition Stand Concept for Regional Administration / Концепт выставочного стенда Воронежской области

A large-scale infrastructure project featuring an interactive exhibition stand developed for the regional administration. Visualization of the plateau with key landmarks and districts of the region.

Масштабный инфраструктурный проект интерактивного выставочного стенда, разработанный для администрации региона. Визуализация плато с ключевыми достопримечательностями и районами области.
