# 3D Modeling & IoT Hardware Concepts Portfolio

## [RU]

Привет! В этом репозитории собрано мое портфолио в области 3D-моделирования (3ds Max) и визуализации концептов устройств. 

Для меня навыки трехмерной графики — это мощный инженерный инструмент. Я использую его для проектирования корпусов физических устройств (IoT), разработки концептов и создания синтетических датасетов для обучения моделей компьютерного зрения (Computer Vision).

### Структура репозитория:

*   📁 [**01-creative-and-art**](./01-creative-and-art) — Личные проекты для души и отработки навыков сложного моделирования. Включает детализированные модели оружия, фотореалистичные интерьеры, экстерьер загородного дома и модель дизайнерских очков.
*   📁 [**02-urban-design-concepts**](./02-urban-design-concepts) — Концептуальный дизайн современного городского мусорного бака. Фокус на эстетику, эргономику и интеграцию в современную городскую архитектуру.
*   📁 [**03-smart-bin-iot-project**](./03-smart-bin-iot-project) — Коммерческий концепт интеллектуальной системы утилизации отходов. Включает несколько детализированных рендеров и подробную техническую презентацию. Устройство спроектировано как комплексное аппаратно-программное решение. Концепт учитывает интеграцию механизма измельчения отходов, управляющего микроконтроллера (ESP32), ультразвуковых датчиков объема и беспроводных модулей связи (Wi-Fi) для реализации сквозной автоматической диспетчеризации и передачи телеметрии на Python-бэкенд.
*   📁 [**04-exhibition-stand-voronezh**](./04-exhibition-stand) — Проект выставочного стенда Воронежской области для администрации области. Визуализация интерактивного плато с детализацией ключевых районов и их достопримечательностей.

## [EN/US]

Welcome! This repository showcases my portfolio in 3D modeling (3ds Max) and hardware/product concept visualization.

I treat 3D graphics as a powerful engineering tool. It allows me to design enclosures for IoT hardware, develop product concepts, and generate synthetic datasets for training Computer Vision models.

### Repository Structure:

*   📁 [**01-creative-and-art**](./01-creative-and-art) — Personal projects focused on advanced hard-surface modeling and interior/exterior design. Features highly detailed weapon models, photorealistic interiors, a countryside house exterior, and a designer eyewear model.
*   📁 [**02-urban-design-concepts**](./02-urban-design-concepts) — Conceptual design of a modern urban waste bin (Type A). Focuses on ergonomics, aesthetics, and seamless integration into modern cityscapes.
*   📁 [**03-smart-bin-iot-project**](./03-smart-bin-iot-project) — A commercial concept of an intelligent waste management system. Includes some high-quality renders and a comprehensive technical presentation. The hardware concept is designed as an integrated cyber-physical system. It incorporates a waste-crushing mechanism, an onboard microcontroller (ESP32), ultrasonic fill-level sensors, and wireless communication modules (Wi-Fi) to enable end-to-end automated fleet dispatching and telemetry transmission to a Python backend.
*   📁 [**04-exhibition-stand-voronezh**](./04-exhibition-stand) — An exhibition stand concept designed for the regional administration. Features a interactive 3D map of the region, highlighting key districts and historical landmarks.
